## How to Use

- **Style prompt construction**: See `suno-field-guide.md` §3 (7-Element Formula, Priority Order) and §5 (Style Field)
- **Chinese-specific tips**: See `suno-chinese-prompts.md` for Mandopop tags, instrument names (guzheng/erhu/pipa/dizi), and tonal pronunciation fixes
- **Chinese industrial production**: See `suno-industrial-handbook.md` Chapters 2-5 for Style tag scoring and Mandarin pop templates
- **Metatag reference**: See `suno-field-guide.md` §4 (Song Structure & Meta Tags) and `blake-crosley-guide.md` (Metatags Reference)
