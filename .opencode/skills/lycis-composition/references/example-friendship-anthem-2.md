# 示例：友情主题炸裂摇滚《同路》

> 用户需求：友情 + 大气 + 炸裂曲风
> 原型匹配：平凡英雄（60%）+ 说书人（40%）
> POV：全程"你"（唯一的那个朋友），Chorus 用"我们"
> 押韵：摇滚体裁，重复 > 韵律，松韵
> 关键约束：**歌词中不出现对话引语**


## 手法分析

| 手法 | 位置 | 说明 |
|------|------|------|
| 物件线 | 通篇 | 荷包蛋 → 方便面 → 沙发 → 蛋，一个贯穿全歌的物质线索 |
| 变形 | V1 | "荷包蛋每次都落在我碗里"——被动语态，不是"你给了"是"落了" |
| 变形 | Pre1 | "那面墙就碎了"——不是"你挡住了"，夸张即真实 |
| 折叠 | V2 | "它记得...比任何账单都清楚"——一张沙发承载整段青春 |
| 留白 | Outro | "蛋 给你留着"——不总结，不给答案，场景定格 |
| 对比 | Chorus | "半碗面" vs "整桌宴席"——小的东西有大的重量 |
| 突破 | Chorus | "这座城市的冬天就冻不住我"——冬天是象征，冻不住是超现实 |
| 隐藏的我 | 通篇 | 全程叙述"你"的好，但"我"的感动全藏在物件里 |

## 共鸣机制标注

- **✅ 共同的敌人**（生活/世界/墙/冬天）——你不是帮我，是一起扛
- **✅ 物质世界的哲学**（鸡蛋/方便面/沙发）——具体物件承载情感
- **✅ 行动的缺口**（"蛋 给你留着"）——半句话，比完整更有力
- **✅ 意象的对比与反差**（半碗面 vs 宴席）——小物的大重量
