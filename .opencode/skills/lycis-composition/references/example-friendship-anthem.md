# 示例：友情主题炸裂摇滚《并肩》

> 用户需求：友情 + 大气 + 炸裂曲风
> 原型匹配：平凡英雄（60%）+ 说书人（40%）
> POV：全程"你"（唯一的那个朋友），Chorus 用"我们"
> 押韵：摇滚体裁，重复 > 韵律，松韵
> 关键约束：**歌词中不出现对话引语**


## 手法分析

| 手法 | 位置 | 行 | 说明 |
|------|------|----|------|
| 变形 | V1 | "灰落了一层又一层" | 静态画面变成动态过程，时间感出来了 |
| 变形 | Pre | "伸手的速度比影子落地还快" | 速度比较变成意象，朋友的速度快过物理规律 |
| 名句 | C | "一个人是根火柴 两个人是场大火" | 反常识+具体物件+尺度跳跃（火柴→大火） |
| 折叠 | V2 | "沙发凹下去一块 那是我的形状" | 一句话包住整段关系史，不解释 |
| 借代 | Outro | "那罐汽水" | 一个物件代表整段友谊，不过期的是等人回来 |
| 暗喻 | C | "你给我的后背 比任何城墙都厚" | 无"像"无"如"的暗喻，后背=信任，城墙=保护 |
| 对比 | C | 火柴/大火 | 个体 vs 两人，尺度跳跃 |
| 留白 | Outro | "等你回来" | 不给答案，汽水等人，听众自己走完 |

## 共鸣机制

- **特异性悖论**：凌晨三点十七分、落灰的拖鞋、过期的汽水
- **隐瞒**：不写"我们是最好的朋友"，用沙发形状、汽水、暗号代替代言
- **脆弱不对称**：Bridge 暴露怕失去——"如果你不在这个城市了"
- **不解决问题**：Outro 不给答案，汽水等人回来
